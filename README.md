# Immobilienrendite-Rechner

Modernes Next.js-Projekt mit TypeScript, App Router und Tailwind CSS fuer die schnelle Analyse von Wohnimmobilien.

## Start

```bash
npm install
npm run dev
```



Falls du `pnpm` verwendest:

```bash
pnpm install
pnpm dev
```

## Funktionen

- Kaufpreis, Nebenkosten, Eigenkapital und Finanzierung anpassen
- Cashflow, Brutto- und Netto-Mietrendite berechnen
- Eigenkapitalrendite inklusive Tilgung und ohne Wertsteigerung einschaetzen
- Break-even-Miete und Finanzierungsbedarf anzeigen
